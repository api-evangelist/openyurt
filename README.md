# OpenYurt (openyurt)
OpenYurt is a CNCF incubating project extending Kubernetes for edge and cloud-edge scenarios. It provides node autonomy during network disconnections, seamless cloud-edge conversion, and unified edge resource management.

## APIs

### OpenYurt Edge Management API
Kubernetes CRDs for edge computing including NodePool, YurtAppSet, YurtAppDaemon, and Raven for cross-edge-cloud networking with YurtHub for local caching and autonomy.
